Projeto U3 — Sistema Livre com Árvore + Ordenação + POO Avançada

Entrega final: 27/11/2025 — Valor: 6 pts

Desenvolva uma aplicação tema livre (ex.: produtos, alunos, agenda) que integre:

Estrutura de dados de árvore (ABB ou AVL).
Algoritmo de ordenação (MergeSort ou HeapSort).
POO avançada: interface(s), herança + polimorfismo, classe genérica, tratamento de exceções (inclui exceção de negócio).
Annotation personalizada lida em tempo de execução (Reflection).
Coleções genéricas (ArrayList / HashMap) quando fizer sentido.

Requisitos obrigatórios

Domínio com classe base + 2 subclasses (polimorfismo ativo). ok
Interfaces (ex.: Repositorio<T>, Ordenacao<T>, Arvore<K,V>). ok
Classe genérica (ex.: RepositorioHash<T> com Function<T,String> para ID). ok (em observação)
Exceções: NegocioException (regra de negócio) + try/catch no Main com mensagens claras. ok 
Annotation @InfoAutor (nome, data) aplicada em ≥2 classes; ler via reflexão e imprimir. ok
Árvore (ABB/AVL): inserir, buscar, travessia em-ordem (retornar valores ordenados pela chave). ok
Ordenação (Merge/Heap): ordenar lista por Comparator. ok
Demonstração no Main: adicionar, listar, buscar, remover, imprimir em ordem (árvore) e ordenado (algoritmo). ok (Ajustes)

Entregáveis

Código-fonte em packages (+ README com como compilar/rodar).
Saída de console mostrando CRUD + árvore + ordenação + leitura de anotação.
